# Automated Podcast Transcription and Topic Segmentation

## Project Overview

This project aims to develop an AI-powered system that automatically transcribes podcast audio recordings into text and segments the transcript into meaningful topics. The system helps users quickly navigate podcast content without listening to the entire episode.

## Problem Statement

Podcast episodes are often lengthy, making it difficult for listeners to locate specific discussions. This project addresses the problem by automatically converting audio into text, identifying topic boundaries, generating summaries, and extracting keywords.

## Objectives

- Convert podcast audio into text using Speech-to-Text technology.
- Identify topic changes within transcripts.
- Generate topic-wise summaries.
- Extract important keywords from each segment.
- Provide an easy-to-use interface for searching and navigating content.
